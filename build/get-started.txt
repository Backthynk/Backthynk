Getting Started with Backthynk
================================

Thank you for downloading Backthynk!

Quick Start:
------------

On Unix/Linux/Mac:
  1. Run: make run
     or: make help (to see all available commands)

On Windows:
  1. Double-click run.bat

The application will start a local server. Check the console output for the URL to access the web interface.

For more information, visit: https://github.com/Fantasim/backthynk

Enjoy!
